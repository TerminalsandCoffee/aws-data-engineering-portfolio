# aws-data-engineering-portfolio
A collection of AWS data engineering projects using EMR, EMR Serverless, S3 data lakes, PySpark, Glue, Athena, and Step Functions. Designed to build real-world ETL pipelines and showcase modern, cost-efficient big data workflows with Terraform and GitHub Actions.
